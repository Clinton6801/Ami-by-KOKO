# French Letter Songs

Place recorded song files here as `[letter]-song.mp3`

Examples: `a-song.mp3`, `b-song.mp3`, etc.

Currently falls back to Web Speech API TTS until native speaker recordings are added.

## Recording Guidelines

- One song per letter (26 total for French alphabet)
- 20–30 seconds each
- Native French speaker pronunciation
- Cheerful, child-friendly melody
- Pattern: Letter name → word → sound → celebration line
