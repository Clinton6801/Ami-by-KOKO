# Yorùbá Number Songs

Place recorded song files here as `[number]-song.mp3`

Examples: `1-song.mp3`, `2-song.mp3`, ..., `10-song.mp3`

Currently falls back to Web Speech API TTS until native speaker recordings are added.

## Recording Guidelines

- One song per number (10 total: 1–10)
- 20–30 seconds each
- Native Yorùbá speaker pronunciation
- Cheerful, child-friendly melody
