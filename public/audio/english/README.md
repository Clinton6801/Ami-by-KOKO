# English Audio Clips

Place pre-recorded native speaker clips here, one per letter:

```

h.mp3  i.mp3  j.mp3  k.mp3  l.mp3  m.mp3  n.mp3
```

Target: < 50KB per file. Format: MP3, mono, 44.1kHz.
Until clips are added, the app falls back to Web Speech API automatically.
