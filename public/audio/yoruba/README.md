# Yorùbá Audio Clips

## Overview
Yorùbá has **25 letters** (NOT 26 like English). This directory contains audio clips for each letter with native speaker pronunciation.

## Letter List (25 total)
```
A B D E Ẹ F G GB H I J K L M N O Ọ P R S Ṣ T U W Y
```

**Note:** C, Q, V, X, Z do not exist in Yorùbá.

## Special Letters
- **Ẹ** (e with dot below) — sounds like "eh" in "bed"
- **Ọ** (o with dot below) — sounds like "aw" in "law"
- **Ṣ** (s with dot below) — sounds like "sh" in "ship"
- **GB** — a digraph (two letters making ONE unique sound)

## File Naming Convention
- Lowercase letter names: `a.mp3`, `b.mp3`, `d.mp3`, etc.
- Special characters: `ẹ.mp3`, `ọ.mp3`, `ṣ.mp3`, `gb.mp3`
- Song files: `a-song.mp3`, `b-song.mp3`, etc. (in `/songs/letters/yoruba/`)

## Recording Specifications
- **Format:** MP3, 128 kbps
- **Sample rate:** 44.1 kHz
- **Duration:** 1–2 seconds per clip
- **Speaker:** Native Yorùbá speaker (preferably from Nigeria)
- **Tone:** Clear, friendly, child-appropriate

## Recording Script
Each clip should follow this pattern:

```
[Letter name] — [Yorùbá word] — [English meaning]
```

### Examples
- **A:** "A — Àgbàdo — corn"
- **Ẹ:** "Ẹ — Ẹkọ — corn pudding"
- **GB:** "GB — Gbàdùn — enjoyment"
- **Ṣ:** "Ṣ — Ṣíṣe — work"

## Tone Marks
Yorùbá uses tone marks extensively. Always include them in the pronunciation:
- **À** (low tone)
- **Á** (high tone)
- **A** (mid tone)

Example: "Àgbàdo" has low-high-low tones.
